Api calculadora de equação de segundo grau.

Sobre o projeto

Esta é uma api que realiza o cálculo de equações do segundo grau.
O projeto foi solicitado pelo grande Mestre de rpg Pedro Alves!


Rota padrão

Por padrão, você será redirecionado para cá, com o tipo de resposta html e status http 200 :)


Endpoints

Abaixo segue lista de enpoints e uma pequena descrição de suas funções.

/resolver-equacao

Este endpoint tem por função realizar o cálculo de uma equação de segundo grau.
Basta enviar um json como body em sua requisição http contendo os seguintes campos:

  • 'valueA': Tipo numérico. Representa o primeiro valor de uma equação do segundo grau.
  • 'valueB': Tipo numérico. Representa o segundo valor de uma equação do segundo grau.
  • 'valueC': Tipo numérico. Representa o terceiro valor de uma equação do segundo grau.

Exemplo de requisição

Endpoint completo: https://quadratic-equation-calculator.herokuapp.com/resolver-equacao

Body: { "valueA": 4, "valueB":-4, "valueC": 1 }
